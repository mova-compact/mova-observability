# MOVA Compact

Evidence-grade dual observability for AI agent runs.

Deterministically correlates **time** and **semantic** event streams into a single, verifiable journal.

## Repository Layout

- `site/` — Astro landing page (marketing + download)
- `docs/` — Product documentation
- `install/` — Platform install & verify scripts
- `checksums/` — SHA256 release checksums
- `examples/` — Quick start guide

## Development

```bash
cd site
npm install
npm run dev     # http://localhost:4321
npm run build   # static output -> site/dist/
```
